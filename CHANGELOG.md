# xyzstyle 项目变更日志

本文档记录 xyzstyle 项目的所有重要变更。

## 版本 0.2.0 (2024-xx-xx)

### 主要变更

- 修复了配置加载功能中的代码问题，确保异常处理正常工作
- 添加了对 Python 3.11 以下版本的兼容性支持（通过添加 tomli 条件依赖）
- 完善了文档配置示例和注释
- 移除了对不存在的样式表文件的引用
- 优化了项目的整体结构和配置文件

### 功能改进

- 增强了配置加载模块的错误处理和日志记录
- 提供了更完整的 doc.toml 配置示例，包括主题选项等
- 改进了 doc/conf.py 文件，添加了详细的配置注释和示例

### 修复的问题

- 修复了 src/xyzstyle/__init__.py 中的代码截断问题
- 修复了缺少 tomllib 导入的问题
- 修复了 theme.toml 中引用不存在文件的问题

## 版本 0.1.0 (2024-xx-xx)

### 初始版本

- 发布了基于 sphinx-book-theme 的 xyzstyle 主题
- 实现了通过 doc.toml 文件进行配置的功能
- 提供了响应式设计和现代化的文档界面
- 添加了侧边注释功能
- 支持自定义静态资源和主题选项配置
